# Voxtral-4B-TTS Colab Setup

This repository contains a Google Colab notebook for running Mistral's Voxtral-4B-TTS model on T4 GPU.

## Quick Start

### Option 1: Open Directly in Colab
Click this link to open the notebook directly in Colab:

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/your-username/voxtral-colab/blob/main/voxtral_colab_setup.ipynb)

### Option 2: Manual Upload
1. Download `voxtral_colab_setup.ipynb`
2. Go to [colab.research.google.com](https://colab.research.google.com)
3. Click "File" → "Upload notebook"
4. Select the downloaded file

## Requirements

- Google Colab with GPU runtime (T4 recommended)
- Internet connection for model download

## Features

- ✅ Fixed BFloat16 compatibility for T4 GPU
- ✅ Optimized memory settings for 16GB VRAM
- ✅ Multilingual TTS support (9 languages)
- ✅ 20 preset voices
- ✅ Voice cloning capability
- ✅ Step-by-step instructions
- ✅ Troubleshooting guide

## Supported Languages

- English, French, Spanish, German, Italian, Portuguese, Dutch, Arabic, Hindi

## Available Voices

- casual_male, casual_female
- professional_male, professional_female
- And 16 more preset voices
